Astuces

Astuces pour une recherche efficace :

•        Utiliser des mots-clés précis et techniques

•        Filtrer par secteur pour réduire le bruit

•        Tester les deux types de recherche : Formation et Métier

Astuces pour explorer le référentiel :

•        Suivre la hiérarchie Secteur → Domaine → Sous-domaine → Spécialité

•        Utiliser les filtres de la page Explorer (secteur, domaine)

•        Consulter les blocs pour voir le détail des compétences visées

Favoris et consultation ultérieure :

•        Sauvegarder les formations intéressantes dans les favoris

•        Télécharger les fiches en PDF pour consultation hors ligne

•        Garder à l’esprit que les favoris sont liés à la session du navigateur

Navigation et résolution de problèmes :

•        Utiliser le fil d’Ariane pour se repérer (secteur > domaine > sous-domaine)

•        Utiliser le bouton Retour du navigateur ou les liens de navigation

•        En cas de résultats vides : élargir les filtres ou modifier les mots-clés

•        En cas de lenteur ou erreur : rafraîchir la page et réessayer plus tard

•        En cas de bug : utiliser le formulaire Contact pour signaler le problème
